T14 - Joint position sensors¶

See also

Joint position sensors

14 x MRE (Magnetic Rotary Encoders) using Hall-effect sensor technology.

Specification¶

12 bit precision, i.e. 4096 values per turn corresponding to about 0.1° precision.
