Project¶

What is a Project¶

Project is the name of the files or directories allowing you to save on your computer:

  • a Behavior and
  • all its potentially attached files (music, C++ module, etc).

How to manage Projects¶

  • The File menu allows you to create, open and save a Project.
  • The Project content panel allows you to manage the files attached to the current project.

To manage the Behavior contained in the project, see Behavior manager panel.

Project available formats¶

You can save / open a Project as:

  • a single compressed file with a CRG extension (ex: MyApplication.crg). The project CRG format is an easy way to archive a project and / or release it to someone else.
  • a directory. The project directory format is a very useful way for collaborative / versioned project whose can be modified by many people.

Note

For further details about saving a project as a CRG file or as a directory, see How to save a project as a CRG and How to save a project as a directory.

How to save a project as a CRG¶

Step Action

Choose File > Save project as....
Make sure the selected file type is CRG File (*.crg).
Browse the destination path and complete the File name field.
Click the Save button.

How to save a project as a directory¶

Step Action

Choose File > Save project as....
Make sure the selected file type is Project directory (Directories).
Select (or create then select) an empty folder.
Click the Choose button.
