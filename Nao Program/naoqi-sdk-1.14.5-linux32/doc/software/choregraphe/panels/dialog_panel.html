Dialog panel - Experimental¶

The Dialog panel allows you to visualize and test verbal interaction with the robot.

Can be used with a simulated robot.

Part Allows you to ...
A Visualize the Q&A flow.
B Type a Human reply.

Options¶

This is an experimental version, if needed, you can deactivate the Dialog view:

  1. Choose Edit > Preferences.
  2. Unselect the Activate dialog widget (Experimental) option.

For further details, see Choregraphe Preferences.
